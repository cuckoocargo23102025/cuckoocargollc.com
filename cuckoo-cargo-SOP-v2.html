Mission
Cung cấp dịch vụ vận chuyển nhanh, tin cậy, cao cấp kết nối cộng đồng người Việt tại Mỹ với quê hương — seamless & trustworthy.
Vision
Trở thành thương hiệu logistics đáng tin cậy nhất cho vận chuyển Mỹ–Việt, được công nhận toàn cầu về premium service.
Positioning
"The premium choice" — nơi luxury service kết hợp logistics efficiency. Không chỉ ship hàng — là cầu nối văn hóa.
Target Audience
Người Việt tại Mỹ · Việt kiều / du học sinh · Người Mỹ tại Los Angeles cần ship hàng về Việt Nam.
Speed — Tốc độ
Thời gian giao hàng siêu tốc vượt tiêu chuẩn ngành. Hàng hóa đến nơi nhanh hơn.
Trust — Tin cậy
Mỗi lô hàng được xử lý cẩn thận, theo dõi realtime, giao hàng với trách nhiệm.
Innovation — Đổi mới
Tận dụng technology để tạo trải nghiệm vận chuyển thông minh, hiệu quả hơn.
Community — Cộng đồng
Được xây dựng bởi và dành cho cộng đồng người Mỹ gốc Việt. Thấu hiểu nhu cầu.

MOBILE FIRST
70%+ khách Việt dùng mobile. CTA "GET A QUOTE" nổi bật thumb-zone. Form đặt đơn 1 cột trên mobile. Floating Zalo button #F47B20 luôn hiển thị.
PERFORMANCE
LCP < 2.5s · CLS < 0.1. Ảnh: WebP format. Font Bebas Neue + Inter preload. Tailwind purge unused CSS. Core Web Vitals: xanh toàn bộ.
DARK / LIGHT MODE
Hero sections: dark bg (#0F0F10) với orange accent. Content sections: white bg. Mix tạo contrast theo brand — không toàn light, không toàn dark.
BILINGUAL LAYOUT
English: Bebas Neue impact. Tiếng Việt: Inter 600 cho heading (Bebas Neue không hỗ trợ dấu tiếng Việt). Consistent spacing giữa 2 ngôn ngữ.
Lưu ý quan trọng: Font Bebas Neue không hỗ trợ dấu tiếng Việt. Dùng Bebas Neue cho headlines tiếng Anh và số. Dùng Inter SemiBold/Bold cho headlines tiếng Việt để giữ đúng brand feel nhưng hiển thị đúng ký tự.

DEV WORKFLOW

Setup Local
git clone → npm install → copy .env.example → docker-compose up → npx prisma migrate dev → npm run seed. MySQL + Redis chạy qua Docker local.
2
Git Branching
main = production · develop = staging · feature/tên-tính-năng = dev branch. PR vào develop → test OK → merge main.
3
Commit Convention
feat: thêm form đặt đơn · fix: sửa pricing calculator · style: cập nhật màu brand #F47B20 · docs: update SOP
4
Pre-merge Checklist
ESLint + Prettier pass · Mobile Safari + Chrome test · Cả 2 ngôn ngữ (vi + en) hiển thị đúng · Brand colors đúng #F47B20 · Lighthouse ≥ 90
5
Code Review
Mỗi PR cần 1 người approve. Checklist: security · performance · mobile responsive · brand consistency (Bebas Neue · #F47B20 · #0F0F10).
