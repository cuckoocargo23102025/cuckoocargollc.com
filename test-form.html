Test: Form → Google Sheet

File này dùng để test kết nối. Xóa sau khi xác nhận hoạt động.

Làm theo 5 bước này trước khi test:

1
Tạo Google Sheet mới tại sheets.google.com. Đặt tên: Cuckoo Cargo Orders. Tạo 5 sheet tab: ORDERS TRACKING CUSTOMERS LABELS SHOPPING
2
Copy Sheet ID từ URL (phần giữa /d/ và /edit). Dán vào ô bên dưới.
3
Mở Extensions → Apps Script trong Google Sheet. Paste toàn bộ nội dung file google-apps-script.js vào đó. Thay PASTE_YOUR_GOOGLE_SHEET_ID_HERE bằng Sheet ID thật.
4
Nhấn Deploy → New deployment → Web App. Execute as: Me. Who has access: Anyone. Copy URL deploy → dán vào ô bên dưới.
5
Điền form bên dưới → nhấn Gửi → kiểm tra Google Sheet có dữ liệu không

Cấu hình kết nối

Bạn phải điền URL này trước. Lấy từ Apps Script → Deploy → Manage deployments.
